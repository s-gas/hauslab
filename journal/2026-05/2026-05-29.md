# 29-05-2026

The gasmetrics app is now divided into different services. Both the gasmetrics-bot and the gasmetrics-server are running on the homelab. Requests can be sent both from Telegram and from the CLI tool. Telegram is used for only posting new readings, whilst the CLI have more features:
- add a reading for today
- add a reading for a specific date
- list the readings, you can also specify how many
- delete a reading
- show the daily average consumption

A next improvement could be to also have the average consuption for a specific period.
