You know what, let's run it locally as a simple binary.

Learned that `host.docker.internal` is the DNS that Docker resolves to the host's IP. I need to set that as target for Prometheus scraping in its `.yaml` file. Also learned that in the `docker-compose.yaml` file, the Prometheus service needs the `extra_hosts` rule set to `"host.docker.internal:host-gateway"`, otherwise it does not work on Linux.
