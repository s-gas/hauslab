Started my homelab journey! 

As my first service, I decided to develop *sysmetrics*, a simple server in Go that exposes CPU and RAM usage. I started with this project to get some confidence with Go.

I used Prometheus and Grafana for my first time and it was pretty fun!

Right now the server, Prometheus and Grafana are all containerized and part of the same structure, but when I will have more services that expose data that I can display, I will detach Prometheus and Grafana and run the Go server natively on the machine, without needing a container for it.

The `gopsutil` is super nice, it was also very interesting understanding how to expose the machine metrics and not the ones for the container. Now I know that these metrics are stored in `/proc`!
